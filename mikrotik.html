مقدمة عن مايكروتك (MikroTik)

مايكروتك (MikroTik) هي شركة لاتفية تأسست عام 1996، متخصصة في تطوير أجهزة وبرمجيات الشبكات. تشتهر الشركة بتقديم حلول شبكات عالية الجودة بأسعار تنافسية، مما جعلها خيارًا شائعًا لمزودي خدمة الإنترنت (ISPs) والشركات الصغيرة والمتوسطة.

منتجات مايكروتك الرئيسية:

  • RouterOS: نظام تشغيل متخصص للشبكات يمكن تثبيته على أجهزة مايكروتك أو على أجهزة الكمبيوتر العادية.
  • RouterBOARD: أجهزة توجيه وتبديل مصممة خصيصًا لتشغيل RouterOS.
  • SwOS: نظام تشغيل مبسط مصمم خصيصًا لمحولات الشبكة (Switches).
  • CRS (Cloud Router Switch): أجهزة تجمع بين وظائف الموجه والمحول.
  • hAP (Home Access Point): نقاط وصول لاسلكية للاستخدام المنزلي والمكتبي.
  • wAP (Wireless Access Point): نقاط وصول لاسلكية للاستخدام الخارجي.
  • LHG (Light Head Grid): أجهزة لاسلكية للاتصالات بعيدة المدى.

نظام التشغيل RouterOS

RouterOS هو نظام التشغيل الأساسي لأجهزة مايكروتك، وهو مبني على نواة Linux. يوفر RouterOS مجموعة واسعة من الميزات والوظائف التي تجعله مناسبًا للاستخدام في مختلف سيناريوهات الشبكات.

التوجيه

دعم لبروتوكولات التوجيه المختلفة مثل RIP، OSPF، BGP، MPLS.

الشبكات اللاسلكية

دعم لمعايير 802.11a/b/g/n/ac، وإنشاء نقاط وصول، وجسور لاسلكية، وشبكات Mesh.

الجدار الناري

جدار ناري متقدم مع دعم لفلترة الحزم، وفحص الحالة، وقوائم العناوين.

NAT

دعم لترجمة عناوين الشبكة (NAT) بأنواعها المختلفة: Source NAT، Destination NAT، Masquerade.

خادم DHCP

خادم DHCP لتوزيع عناوين IP تلقائيًا، مع دعم للتأجير الثابت والديناميكي.

إدارة المستخدمين

نظام إدارة مستخدمين متقدم مع دعم للمصادقة المحلية وRADIUS.

إدارة النطاق الترددي

أدوات متقدمة لإدارة النطاق الترددي وجودة الخدمة (QoS).

VPN

دعم لبروتوكولات VPN المختلفة مثل IPsec، OpenVPN، PPTP، L2TP، SSTP.

طرق الوصول إلى RouterOS

يوفر RouterOS عدة طرق للوصول إليه وإدارته:

1. Winbox

أداة رسومية تعمل على نظام Windows، توفر واجهة سهلة الاستخدام لإدارة أجهزة مايكروتك.

المميزات:

  • واجهة رسومية سهلة الاستخدام
  • يمكن الاتصال عبر عنوان IP أو عنوان MAC
  • يدعم معظم وظائف RouterOS
  • لا يحتاج إلى تثبيت (يمكن تشغيله مباشرة)

2. WebFig

واجهة ويب تسمح بإدارة الجهاز من خلال متصفح الويب.

المميزات:

  • لا يحتاج إلى تثبيت برامج إضافية
  • يعمل على جميع أنظمة التشغيل
  • يدعم معظم وظائف RouterOS
  • يمكن الوصول إليه من أي جهاز متصل بالشبكة

3. واجهة سطر الأوامر (CLI)

واجهة نصية تسمح بإدارة الجهاز من خلال الأوامر النصية.

المميزات:

  • يمكن الوصول إليها عبر SSH، Telnet، أو المنفذ التسلسلي
  • مناسبة للإدارة عن بعد وأتمتة المهام
  • تدعم جميع وظائف RouterOS
  • أداء أفضل في الاتصالات البطيئة

4. تطبيق الهاتف المحمول

تطبيق للهواتف الذكية يسمح بإدارة أساسية للجهاز.

المميزات:

  • متاح لأنظمة Android و iOS
  • يوفر وظائف أساسية للمراقبة والإدارة
  • مناسب للاستخدام السريع والمراقبة
  • واجهة بسيطة وسهلة الاستخدام

التكوين الأساسي لجهاز MikroTik

في هذا القسم، سنتعرف على الخطوات الأساسية لتكوين جهاز MikroTik جديد.

الخطوة 1: الاتصال بالجهاز

عند تشغيل جهاز MikroTik جديد، يمكنك الاتصال به بإحدى الطرق التالية:

  • عبر Winbox: استخدم Winbox للاتصال بعنوان MAC للجهاز (يظهر في قائمة الأجهزة المكتشفة).
  • عبر الشبكة: قم بتوصيل جهاز الكمبيوتر بالمنفذ ether1 واستخدم عنوان IP الافتراضي للجهاز (عادة 192.168.88.1).

بيانات الدخول الافتراضية:

  • اسم المستخدم: admin
  • كلمة المرور: (فارغة)

الخطوة 2: تغيير كلمة المرور

من الضروري تغيير كلمة المرور الافتراضية لتأمين الجهاز:

/system user set 0 password=كلمة_المرور_الجديدة

أو من خلال Winbox: System > Users > انقر نقرًا مزدوجًا على المستخدم admin > غير كلمة المرور.

الخطوة 3: تكوين الواجهات (Interfaces)

تسمية الواجهات وتكوينها حسب الاستخدام:

/interface set 0 name=WAN comment="اتصال الإنترنت"
/interface set 1 name=LAN comment="الشبكة المحلية"

أو من خلال Winbox: Interfaces > انقر نقرًا مزدوجًا على الواجهة > غير الاسم والتعليق.

الخطوة 4: تكوين عناوين IP

تعيين عناوين IP للواجهات:

# تكوين واجهة WAN للحصول على عنوان IP من مزود خدمة الإنترنت
/ip dhcp-client add interface=WAN disabled=no

# تكوين واجهة LAN بعنوان IP ثابت
/ip address add address=192.168.1.1/24 interface=LAN

أو من خلال Winbox: IP > Addresses > + > أدخل العنوان والواجهة.

الخطوة 5: تكوين خادم DHCP

إعداد خادم DHCP لتوزيع عناوين IP على الأجهزة في الشبكة المحلية:

# إنشاء مجمع عناوين DHCP
/ip pool add name=dhcp-pool ranges=192.168.1.2-192.168.1.254

# إنشاء شبكة DHCP
/ip dhcp-server network add address=192.168.1.0/24 gateway=192.168.1.1 dns-server=8.8.8.8,8.8.4.4

# إنشاء خادم DHCP
/ip dhcp-server add name=dhcp-server interface=LAN address-pool=dhcp-pool disabled=no

# تمكين خادم DHCP
/ip dhcp-server enable 0

أو من خلال Winbox: IP > DHCP Server > DHCP > + > أدخل الإعدادات.

الخطوة 6: تكوين NAT

إعداد NAT للسماح للأجهزة في الشبكة المحلية بالوصول إلى الإنترنت:

/ip firewall nat add chain=srcnat action=masquerade out-interface=WAN

أو من خلال Winbox: IP > Firewall > NAT > + > أدخل الإعدادات.

الخطوة 7: تكوين قواعد الجدار الناري الأساسية

إعداد قواعد الجدار الناري الأساسية لحماية الشبكة:

# السماح بالاتصالات المعروفة والمرتبطة
/ip firewall filter add chain=forward action=accept connection-state=established,related

# حظر الاتصالات غير المصرح بها من الإنترنت إلى الشبكة المحلية
/ip firewall filter add chain=forward action=drop in-interface=WAN connection-state=new connection-nat-state=!dstnat

أو من خلال Winbox: IP > Firewall > Filter Rules > + > أدخل الإعدادات.

الخطوة 8: حفظ التكوين

حفظ التكوين لضمان استمراره بعد إعادة تشغيل الجهاز:

/system backup save name=initial-config

أو من خلال Winbox: System > Backup > Backup > أدخل اسم النسخة الاحتياطية.

تكوين الشبكة اللاسلكية

إذا كان جهاز MikroTik يدعم الشبكات اللاسلكية، يمكنك تكوين نقطة وصول لاسلكية كما يلي:

الخطوة 1: تكوين الواجهة اللاسلكية

# تعيين وضع نقطة الوصول
/interface wireless set 0 mode=ap-bridge

# تعيين اسم الشبكة (SSID)
/interface wireless set 0 ssid=MyNetwork

# تعيين التردد
/interface wireless set 0 frequency=2437 (للقناة 6)

# تعيين عرض القناة
/interface wireless set 0 channel-width=20/40mhz-Ce

# تمكين الواجهة اللاسلكية
/interface wireless enable 0

الخطوة 2: تكوين أمان الشبكة اللاسلكية

# تعيين وضع الأمان (WPA2-PSK)
/interface wireless security-profiles add name=profile1 mode=dynamic-keys authentication-types=wpa2-psk unicast-ciphers=aes-ccm group-ciphers=aes-ccm wpa2-pre-shared-key=كلمة_المرور

# تطبيق ملف الأمان على الواجهة اللاسلكية
/interface wireless set 0 security-profile=profile1

الخطوة 3: إضافة الواجهة اللاسلكية إلى جسر الشبكة المحلية

# إنشاء جسر
/interface bridge add name=bridge1

# إضافة الواجهات إلى الجسر
/interface bridge port add bridge=bridge1 interface=LAN
/interface bridge port add bridge=bridge1 interface=wlan1

# تعيين عنوان IP للجسر
/ip address add address=192.168.1.1/24 interface=bridge1

مراقبة وإدارة الشبكة

يوفر RouterOS أدوات متقدمة لمراقبة وإدارة الشبكة:

1. مراقبة استخدام النطاق الترددي

يمكنك مراقبة استخدام النطاق الترددي لكل واجهة أو لكل مستخدم:

# عرض إحصائيات الواجهات
/interface monitor-traffic WAN,LAN

# إنشاء قواعد لمراقبة حركة المرور
/ip traffic-flow set enabled=yes

2. إدارة النطاق الترددي (QoS)

يمكنك تحديد أولويات حركة المرور وتقييد النطاق الترددي:

# إنشاء قواعد علامات حركة المرور
/ip firewall mangle add chain=forward protocol=tcp dst-port=80,443 action=mark-connection new-connection-mark=web-con
/ip firewall mangle add chain=forward connection-mark=web-con action=mark-packet new-packet-mark=web-traffic

# إنشاء قواعد إدارة النطاق الترددي
/queue simple add name=web-queue packet-marks=web-traffic max-limit=5M/5M

3. جدولة المهام

يمكنك جدولة المهام لتنفيذها في أوقات محددة:

# إعادة تشغيل الجهاز كل يوم في الساعة 3 صباحًا
/system scheduler add name=daily-reboot on-event="/system reboot" start-time=03:00:00 interval=1d

4. النسخ الاحتياطي والاستعادة

يمكنك إنشاء نسخ احتياطية للتكوين واستعادتها عند الحاجة:

# إنشاء نسخة احتياطية
/system backup save name=backup-date

# استعادة نسخة احتياطية
/system backup load name=backup-date

الشهادات المهنية من MikroTik

تقدم MikroTik مجموعة من الشهادات المهنية للمتخصصين في مجال الشبكات:

MTCNA (MikroTik Certified Network Associate)

الشهادة الأساسية التي تغطي المفاهيم الأساسية لأجهزة MikroTik وRouterOS.

المواضيع:

  • أساسيات RouterOS
  • التوجيه
  • الجدار الناري
  • الشبكات اللاسلكية
  • إدارة النطاق الترددي

MTCRE (MikroTik Certified Routing Engineer)

شهادة متقدمة تركز على التوجيه في RouterOS.

المواضيع:

  • التوجيه الثابت
  • OSPF
  • BGP
  • MPLS
  • VRF

MTCWE (MikroTik Certified Wireless Engineer)

شهادة متخصصة في الشبكات اللاسلكية باستخدام أجهزة MikroTik.

المواضيع:

  • أساسيات الشبكات اللاسلكية
  • تكوين نقاط الوصول
  • الجسور اللاسلكية
  • شبكات Mesh
  • استكشاف الأخطاء وإصلاحها

MTCTCE (MikroTik Certified Traffic Control Engineer)

شهادة متخصصة في إدارة النطاق الترددي وجودة الخدمة.

المواضيع:

  • علامات حركة المرور
  • قواعد الطوابير
  • إدارة الازدحام
  • تشكيل حركة المرور
  • مراقبة الأداء
